# Amazon Clone – Frontend UI

A responsive **Amazon homepage clone** built using **HTML5** and **CSS3**. This project recreates the look and feel of Amazon's landing page, focusing on responsive design, modern CSS layouts, and clean user interface development.

---

## Project Overview

The goal of this project was to strengthen front-end web development skills by replicating the design of one of the world's largest e-commerce platforms. The project demonstrates proficiency in HTML structure, CSS styling, Flexbox layouts, and responsive web design principles.

---

## Features

- Responsive Amazon-inspired homepage
- Navigation bar with logo, search bar, and menu options
- Hero banner section
- Product category cards
- Shopping sections with multiple product boxes
- Footer similar to Amazon
- Clean and organized layout using CSS Flexbox
- Responsive design for different screen sizes

## How to Run

1. Clone the repository

```bash
git clone https://github.com/shivi2103/amazon-clone.git
```

2. Open the project folder

```bash
cd amazon-clone
```

3. Open `index.html` in your browser.

No additional setup or installation is required.
